# 数据库设计

## 表结构说明

### 表1：产品表（product）
| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键 |
| anti_fake_code | VARCHAR(20) | 防伪码，唯一 |
| name | VARCHAR(100) | 食品名称 |
| specification | VARCHAR(50) | 规格 |
| batch_number | VARCHAR(50) | 生产批次 |
| production_date | DATE | 生产日期 |
| shelf_life | VARCHAR(50) | 保质期 |
| image_url | VARCHAR(255) | 产品图片URL |
| contact_phone | VARCHAR(20) | 采购联系电话 |
| contact_email | VARCHAR(100) | 采购联系邮箱 |

### 表2：原料采购表（material_purchase）
| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键 |
| product_id | BIGINT | 外键，关联产品 |
| material_name | VARCHAR(100) | 原料名称 |
| producer_name | VARCHAR(100) | 生产者 |
| producer_address | VARCHAR(255) | 生产者地址 |

### 表3：贮存表（storage）
| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键 |
| product_id | BIGINT | 外键 |
| storage_time | DATETIME | 入库时间 |
| outbound_time | DATETIME | 出库时间 |
| quantity | DOUBLE | 数量 |
| unit | VARCHAR(20) | 单位 |

### 表4：出厂检验表（inspection）
| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键 |
| product_id | BIGINT | 外键 |
| sample_name | VARCHAR(100) | 抽检样品名称 |
| sample_quantity | INT | 样品数量 |
| sample_specification | VARCHAR(100) | 样品规格 |

### 表5：储运销售表（transport_sale）
| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键 |
| product_id | BIGINT | 外键 |
| environment_temperature | DOUBLE | 环境温度 |
| product_temperature | DOUBLE | 产品温度 |
| time | DATETIME | 时间 |

### 表6：投诉表（complaint）
| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键 |
| product_id | BIGINT | 外键 |
| complaint_reason | VARCHAR(500) | 投诉原因 |
| complaint_time | DATETIME | 投诉时间 |
